# 电商网站登录模块测试用例

## 一、等价类划分分析

### 1.1 用户名等价类

| 等价类 | 说明 | 代表值 |
|--------|------|--------|
| 有效等价类 | 4-16位字母/数字/下划线 | test001 |
| 无效等价类1 | 少于4位 | ab |
| 无效等价类2 | 超过16位 | abcdefghijklmnopq |
| 无效等价类3 | 含特殊字符 | test@001 |
| 无效等价类4 | 为空 | （空） |

### 1.2 密码等价类

| 等价类 | 说明 | 代表值 |
|--------|------|--------|
| 有效等价类 | 6-20位，含字母+数字 | Test123 |
| 无效等价类1 | 少于6位 | 12345 |
| 无效等价类2 | 超过20位 | 123456789012345678901 |
| 无效等价类3 | 纯数字 | 123456 |

| 测试值 | 说明 | 预期结果 |
|--------|------|---------|
| 3位 | min-1 | 提示"用户名长度不符合要求" |
| 4位 | min | 可正常登录 |
| 5位 | min+1 | 可正常登录 |
| 15位 | max-1 | 可正常登录 |

| 17位 | max+1 | 提示"用户名长度不符合要求" |
